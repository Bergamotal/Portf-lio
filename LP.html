A linguagem de programação é um método padronizado, formado por um conjunto de regras sintáticas e semânticas, de implementação de um código fonte - que pode ser compilado e transformado em um programa de computador, ou usado como script interpretado que informará instruções de processamento ao computador.



-- Regras Sintáticas & Regras Semânticas --


- Regras Sintáticas são as regras que caem em cima das palavras(alguma palavra escrita de forma incorreta no código);



-Regras Semânticas são as regras que caem em cima da frase(algum linha no código que foi escrita de forma incorreta).



-- Variáveis --



Como pode-se ver no exemplo, as variáveis são como as vagas em um estacionamento, onde podem assumir uma numeração, nome, tipo(palavra reservada) ou tamanhos diferentes.






-- Definição de Variáveis --


A variável ao ser declarada vai passar a existir na programação, fazendo com que ela não precise ser declarada novamente e onforme o tipo da variável é possível atribuir um valor a ela.




Ou valor da variável pode ser atribuída mais tarde no código.

-- Estruturas de Decisão --


As estruturas de decisões são lógicas que realizam os blocos de instruções dentro da estrutura(como if & else)



-- If & Else --


A estrutura de decisão if & else(se & senão) abre um teste lógico onde "se" a condição for verdadeira o bloco de instruções dentro do if é ativado e vai permitir que continue rodando as linhas do código dentro do if, "senão" for verdadeira o else é ativado e o que está dentro dele será realizado(caso else for utilizado).





-- Switch & Case & Defaut --


O switch é uma estrutura que permite ter vários caminhos no fluxo do programa dependendo do seu valor, e caso o valor não atenda a nenhuma das opções o default é acionado dentro switch, tendo uma case para cada caminho, onde cada um deles pode ser quebrado usando um break.

-- Estrutura de Repetição --


As estruturas de repetição são lógicas que repetem os blocos de instruções dentro da estrutura(como for ou while).



-- While --

A estrutura de repetição while permite a criação de um loop que executa um bloco de instruções repetidas vezes enquanto uma condição for verdadeira.



-- Do / While --

Semelhante ao while o do/while obriga que o bloco de instruções seja executado ao menos uma vez.



-- For --

Possui a mesma ideia que o while porém com uma estrutura fixa que obriga a criação de uma variável de controle.






-- Vetores --


Vetores (array) são variáveis que reservam regiões da memória que podem alocar diversos valores do mesmo tipo em sequencia. Para acessar estes valores utiliza-se o nome dado ao vetor + um índice numérico.






Na imagem, temos a representação de um vetor chamado **num** de 3 posições. Nesta representação temos o seguinte:

- Posição 0 → valor 5

- Posição 1 → valor 8

- Posição 2 → valor 4





-- Matriz --


Matrizes são vetores N dimensionais, visto que um vetor é unidimensional.






O funcionamento do índice é o mesmo dos vetores, a diferença é que agora os índice localizam os dados em N dimensões






-- Funções --


Funções são blocos de instruções que são executados somente quando “chamados”. Funções pode ter parâmetros, valores que são passados durante sua chamada. Funções também podem ter retorno, um valor que é retornado para o local de chamada da função.






-- Função Recursiva --


São funções que executam a chamada delas mesmas dentro de seu próprio bloco de instruções.
